# Тесты API Stellar Burgers

Автотесты для тестирования API сервиса Stellar Burgers.

## Установка

```bash
mvn clean install
```

## Запуск тестов

```bash
mvn clean test
```

## Отчёт Allure

```bash
mvn allure:serve
```

## Что тестируется

- Регистрация пользователей
- Авторизация пользователей  
- Создание заказов

## Технологии

Java 11, Maven, JUnit, REST Assured, Allure
